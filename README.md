# Sistema Municipal de Gestão de Vagas - Educação Infantil

Sistema web institucional para gerenciamento de demanda, inscrições e alocação de vagas em creches e pré-escolas da rede municipal.

## Sobre

Substitui planilhas e cadastros manuais, mantendo base única municipal com transparência, rastreabilidade e conformidade administrativa.

## Recursos Principais

- Registro presencial de inscrições por instituição
- Parametrização anual de critérios e pesos
- Cálculo automático de pontuação
- Lista de espera unificada por modalidade
- Gestão de alocação e matrícula
- Sistema completo de auditoria e logs

## Usuários do Sistema

- **Secretaria Municipal de Educação**: Parametrização, cadastro de instituições e visão global
- **Diretor/Secretário**: Inscrições, turmas e alocação na própria unidade
- **Administrador**: Gestão técnica do sistema

> Responsáveis não possuem acesso. Inscrições são exclusivamente presenciais.

## Fluxo do Processo

1. Inscrição presencial na instituição
2. Classificação automática por pontuação
3. Lista de espera ordenada
4. Convocação e alocação de vaga
5. Matrícula com validação documental
