title 7.1 — Topologia de Produção (Diagrama de Componentes)

skinparam componentStyle rectangle

node "Servidor de Produção\n(Ubuntu 22.04)" {

  component "Nginx\n(TLS 1.2+ / Proxy Reverso)" as Nginx
  component "Gunicorn\n(WSGI - Workers 2–4)" as Gunicorn
  component "Aplicação Flask\n(SICREM)" as FlaskApp
  component "Systemd Service\n(sicrem)" as Systemd

  database "PostgreSQL 14\n(sicrem_prod)" as DB

}

node "Infraestrutura Externa" {
  component "Servidor SMTP\n(E-mail TLS)" as SMTP
}

actor "Usuário (Operador)" as User

User --> Nginx : HTTPS (443)
Nginx --> Gunicorn : Proxy (Unix Socket / HTTP)
Gunicorn --> FlaskApp : WSGI
FlaskApp --> DB : SQL (psycopg2)
FlaskApp --> SMTP : SMTP/TLS

Systemd --> Gunicorn : Gerencia processo
Systemd --> FlaskApp : Inicialização (wsgi:app)

@enduml
