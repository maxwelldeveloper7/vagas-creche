@startuml

actor "Operador\n(Diretor/Secretário)" as Operador

participant "Interface Web\n(Template HTML)" as Interface
participant "Servidor Flask\n(Controlador de Inscrição)" as Flask
database "Banco de Dados\nPostgreSQL" as DB

Operador -> Interface : Clica em "Conferir Inscrição"

Interface -> Flask : POST /conferir_inscricao\n(dados temporários da inscrição)

Flask -> Flask : Validar dados do responsável
Flask -> Flask : Validar dados da criança
Flask -> Flask : Validar regras de negócio

Flask -> DB : Consultar dados relacionados\n(criança, responsável)

DB --> Flask : Retorna dados consultados

Flask -> Flask : Montar objeto resumo da inscrição

Flask --> Interface : Renderizar página\nResumo da Inscrição

Interface --> Operador : Exibir formulário institucional\ncom dados da inscrição

Operador -> Interface : Confirmar inscrição

Interface -> Flask : POST /confirmar_inscricao

Flask -> DB : Persistir dados da inscrição

DB --> Flask : Confirma gravação

Flask --> Interface : Redirecionar para geração\ndo comprovante

@enduml
