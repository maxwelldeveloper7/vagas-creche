@startuml

actor "Operador (Diretor/Secretário)" as Operador
participant "Navegador Web" as Browser
participant "Servidor Flask" as Flask
database "PostgreSQL" as DB

Operador -> Browser : acessar tela de cadastro de criança
Browser -> Flask : GET /crianca/nova

Flask -> DB : consultar responsável associado
DB --> Flask : dados do responsável

Flask --> Browser : renderizar formulário de cadastro

Operador -> Browser : preencher dados da criança
Browser -> Flask : POST /crianca/salvar\n(dados da criança)

Flask -> Flask : validar dados do formulário

alt CPF inválido ou campos obrigatórios faltando
    Flask --> Browser : retornar mensagem de erro
else Dados válidos
    Flask -> DB : verificar duplicidade CPF criança
    DB --> Flask : resultado da consulta
    
    alt CPF já cadastrado
        Flask --> Browser : informar duplicidade
    else CPF não cadastrado
        Flask -> DB : inserir registro na tabela CRIANCAS
        DB --> Flask : confirmação de inserção
        
        Flask --> Browser : exibir confirmação de cadastro
        
        Flask --> Browser : perguntar\n"Deseja cadastrar outra criança?"
    end
end

@enduml
