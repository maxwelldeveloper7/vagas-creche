@startuml

actor "Operador (Diretor/Secretário)" as Operador
participant "Interface Web (HTML)" as UI
participant "Servidor Flask" as Flask
participant "Banco de Dados PostgreSQL" as DB
participant "Gerador de PDF" as PDF

Operador -> UI : Confirmar dados da inscrição
UI -> Flask : POST /salvar-inscricao

Flask -> DB : Verificar duplicidade de inscrição
DB --> Flask : Resultado da consulta

alt inscrição já existente
    Flask -> UI : Exibir mensagem de erro
else inscrição válida
    Flask -> DB : Inserir registro da inscrição
    DB --> Flask : Confirmação + número da inscrição

    Flask -> PDF : Gerar comprovante PDF
    PDF -> PDF : Criptografar PDF (CPF da criança)

    PDF --> Flask : Arquivo PDF gerado

    Flask -> UI : Retornar comprovante
    UI -> Operador : Exibir / permitir download
end

@enduml
