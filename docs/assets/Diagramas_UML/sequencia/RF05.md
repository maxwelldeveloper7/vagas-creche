@startuml

actor "Operador\n(Diretor/Secretário)" as Operador

participant "Interface Web\n(HTML/Flask Template)" as Interface
participant "Servidor Flask\n(Controlador)" as Flask
database "PostgreSQL\nBanco de Dados" as DB

== Início do Cadastro ==

Operador -> Interface : Acessar tela de cadastro de responsável
Interface -> Flask : Requisição HTTP GET /responsavel/novo

Flask -> Interface : Renderizar formulário de cadastro

== Preenchimento do Formulário ==

Operador -> Interface : Preencher dados do responsável\n(nome, CPF, telefone, endereço...)

Operador -> Interface : Submeter formulário

Interface -> Flask : HTTP POST /responsavel/salvar\n(dados do formulário)

== Validação ==

Flask -> Flask : Validar campos obrigatórios
Flask -> Flask : Validar formato do CPF

alt CPF já cadastrado
    Flask -> Interface : Exibir mensagem\n"Responsável já cadastrado"
else CPF válido e não existente
    Flask -> DB : INSERT responsavel
    DB -> Flask : Confirma gravação

    Flask -> Interface : Exibir confirmação de cadastro
end

== Continuação do Fluxo ==

Interface -> Operador : Exibir sucesso e continuar inscrição

@enduml
