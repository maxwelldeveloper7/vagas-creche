@startuml

actor Usuario

participant "Interface Web\n(Navegador)" as Web
participant "Servidor Flask" as Flask
database "PostgreSQL" as DB

Usuario -> Web : Acessa tela de consulta
Web -> Flask : GET /consultar-inscricoes

Flask -> Web : Renderiza página de consulta

Usuario -> Web : Informa filtros\n(nome / CPF / nº inscrição)

Web -> Flask : POST /consultar-inscricoes\n(filtros)

Flask -> Flask : Validar dados da consulta

alt Usuário é Administrador
    Flask -> DB : Consultar todas inscrições
else Usuário é Diretor ou Secretário
    Flask -> DB : Consultar inscrições da unidade
end

DB --> Flask : Retorna registros

Flask -> Flask : Processar resultados

Flask -> Web : Renderizar página com resultados

Web -> Usuario : Exibir lista de inscrições

Usuario -> Web : Selecionar inscrição

Web -> Flask : GET /inscricao/{id}

Flask -> DB : Buscar dados completos da inscrição

DB --> Flask : Retornar dados

Flask -> Web : Renderizar página de detalhes

Web -> Usuario : Exibir dados completos

@enduml
