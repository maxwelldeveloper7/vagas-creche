@startuml

actor Usuario

participant "Interface Web\n(Login)" as UI
participant "Servidor Flask\n(Controlador de Autenticação)" as Flask
database "Banco de Dados\n(PostgreSQL)" as DB

Usuario -> UI : acessar tela de login

UI -> Usuario : exibir formulário de login

Usuario -> UI : informar email e senha

UI -> Flask : enviar credenciais (POST /login)

Flask -> DB : consultar usuário por email

DB --> Flask : retornar dados do usuário

alt Credenciais válidas

    Flask -> Flask : verificar hash da senha
    
    Flask -> UI : autenticação bem-sucedida
    UI -> Usuario : redirecionar para painel do sistema

else Credenciais inválidas

    Flask -> UI : erro de autenticação
    UI -> Usuario : exibir mensagem de erro

end

@enduml
