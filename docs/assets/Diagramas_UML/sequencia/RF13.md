@startuml

actor Administrador

participant "Interface Web\n(Template HTML)" as UI
participant "Servidor Flask\n(Controlador)" as Flask
participant "Módulo de Relatórios" as RelatorioService
database "PostgreSQL" as DB

Administrador -> UI : acessar menu Relatórios

UI -> Flask : GET /relatorios

Flask -> UI : renderizar página de filtros

Administrador -> UI : selecionar filtros\n(tipo de relatório, período, unidade)

UI -> Flask : POST /relatorios/gerar

Flask -> RelatorioService : gerarRelatorio(filtros)

RelatorioService -> DB : consultar inscrições\ncom filtros aplicados

DB --> RelatorioService : retornar dados

RelatorioService -> RelatorioService : processar dados\n(formatar relatório)

RelatorioService --> Flask : relatório gerado

Flask -> UI : renderizar tabela de resultados

Administrador -> UI : solicitar exportação CSV

UI -> Flask : GET /relatorios/exportar

Flask -> RelatorioService : exportarCSV(dados)

RelatorioService --> Flask : arquivo CSV

Flask -> UI : download do arquivo CSV

@enduml
