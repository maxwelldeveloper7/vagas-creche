@startuml

actor "Operador\n(Diretor ou Secretário)" as Operador
participant "Navegador Web" as Browser
participant "Servidor Flask" as Flask
database "PostgreSQL" as DB

Operador -> Browser : Acessa tela "Nova Inscrição"

Browser -> Flask : GET /nova-inscricao

Flask -> DB : Verificar sessão do usuário
DB --> Flask : Sessão válida

Flask --> Browser : Renderiza formulário\npara informar CPF da criança

Operador -> Browser : Digita CPF da criança\n e envia formulário

Browser -> Flask : POST /verificar-cpf

Flask -> DB : Consultar inscrição existente\npor CPF da criança

alt Inscrição já existe
    DB --> Flask : Registro encontrado
    Flask --> Browser : Exibir mensagem\n"Inscrição já existente"
else Inscrição não existe
    DB --> Flask : Nenhum registro encontrado
    Flask --> Browser : Abrir formulário\nCadastro da Criança\n(CPF preenchido automaticamente)
end

@enduml
