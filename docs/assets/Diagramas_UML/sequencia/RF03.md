@startuml

actor Administrador

participant "Interface Web\n(Navegador)" as Web
participant "Servidor Flask" as Flask
database "PostgreSQL" as DB

Administrador -> Web : acessar tela de cadastro de usuário
Web -> Flask : GET /usuarios/novo

Flask -> Web : renderizar formulário de cadastro

Administrador -> Web : preencher dados do usuário
Web -> Flask : POST /usuarios\n(dados do usuário)

Flask -> Flask : validar dados

alt dados válidos
    Flask -> DB : INSERT INTO usuarios
    DB --> Flask : confirmação

    Flask -> Web : redirecionar para lista de usuários
    Web -> Administrador : exibir mensagem "Usuário cadastrado com sucesso"

else dados inválidos
    Flask -> Web : retornar formulário com erros de validação
    Web -> Administrador : exibir mensagens de erro
end

@enduml
